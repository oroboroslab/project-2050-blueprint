## Running the page

Open `index.html` directly in any modern browser, or serve the folder:

```
cd project-2050-blueprint
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.

## Design

- Color palette: Royal Blue (`#002366`) on White (`#ffffff`), with accent Deep Azure (`#0b3d91`) and Gold trim (`#c9a227`) for classification strips.
- Advertisement cards: large globe SVG protrudes off the card top and bottom; four mini ouroboros logos anchor the corners.
- Typography: modern sans-serif (Inter / system-ui) for headings, serif (Newsreader) for briefing body.
